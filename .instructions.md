## 🎨 Design System

### Colors (Tailwind Sky Palette)

Primary color scheme uses **sky colors** for a light, modern touch:

- Sky-50: Lightest background
- Sky-100, Sky-200, Sky-300: Light accents
- Sky-400, Sky-500, Sky-600: Action buttons, primary CTA
- Sky-700: Medium text/accents
- Sky-800, Sky-900: Dark text, headers
- Sky-950: Dark backgrounds (Contact section)

### Fonts

- **Playfair Display:** Headlines, elegant titles, slogan
- **Poppins:** Body text, all descriptions, UI elements
- Both fonts loaded from Google Fonts

### Interactive Yoga Poses 🧘‍♀️

The site features **minimalist SVG yoga poses** throughout for visual interest and interactivity:

#### Pose Types Included:

1. **Tree Pose (Vrksasana)** - Balance and stability
2. **Downward Dog (Adho Mukha Svanasana)** - Strength and core
3. **Lotus (Padmasana)** - Meditation and flexibility
4. **Warrior I (Virabhadrasana I)** - Power and energy
5. **Child's Pose (Balasana)** - Relaxation and recovery
6. **Triangle (Trikonasana)** - Full body stretch

#### Usage Throughout Site:

- **Hero Section:** Animated floating silhouettes (Tree & Lotus) with breathing/float animations
- **Services Section:** Each service card displays a matching yoga pose icon on hover
- **Contact Section:** Subtle warrior and lotus pose silhouettes in dark background

#### Animations:

- `pose-float`: Smooth floating animation (4s cycle)
- `pose-breathe`: Breathing pulse effect (3s cycle)
- `pose-float-delayed`: Float with 1s delay for staggered visual effect
- Hover reveal with opacity transitions for smooth appearance

#### Component File:

- `src/components/YogaPoses.tsx` - Contains all SVG pose components and the `YogaPoseIcon` helper
- All poses use `stroke` and `fill` properties for clean, minimalist appearance

## 🔧 Development Tips

### Important Patterns

1. **Section Structure**

   ```tsx
   <section
     id="services"
     className="py-24 px-4 bg-gradient-to-b from-white to-sky-50"
   >
     <div className="max-w-7xl mx-auto">{/* content */}</div>
   </section>
   ```

2. **Font Usage**
   - Playfair for headings: `font-playfair`
   - Poppins for body: `font-poppins`

3. **Color Usage**
   - Sky colors: `sky-50` through `sky-950`
   - Backgrounds: `sky-50`, `sky-100`, `sky-200`
   - Text: `sky-700`, `sky-900`
   - Accents: `sky-400`, `sky-600`

### Common Tailwind Classes

```
Text colors: text-sky-700, text-sky-900
Backgrounds: bg-sky-50, bg-gradient-to-b from-white to-sky-50
Borders: border border-sky-100, hover:border-sky-300

Hover effects: hover:shadow-2xl, hover:scale-105
```
